# LlamaWhisperer

LlamaWhisperer is a macOS menu bar application that provides speech-to-text functionality using Whisper and Ollama. It allows you to quickly record audio, transcribe it, and paste the results directly into any application.

## Features

• Quick audio recording with keyboard shortcut (Cmd+Shift+D)
• Automatic transcription using Whisper
• Optional text cleaning with Ollama for improved punctuation and formatting
• Clean, intuitive menu bar interface
• No persistent history storage (focus on real-time transcription)

## Requirements

• macOS 12.0 or later
• Ollama installed and running
• A language model (like llama3.2:3b) downloaded via Ollama

## Installation

### Prerequisites

1. Install Ollama on your Mac
2. Download a language model:

```ollama pull llama3.2:3b```
